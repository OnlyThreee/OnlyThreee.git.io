事前准备

下载 jQuery

下载 popper

官网：https://popper.js.org/

下载 BootStrap4

官网：http://www.getbootstrap.com

如下图所示，点击Dowmload按钮下载压缩包，此页面下还介绍了其他多种下载方式，供您选择最适合的方式。压缩包内含有 css 和 js 两个文件夹，将其解压出来，建议新建一个文件夹存放，以便后期使用。
